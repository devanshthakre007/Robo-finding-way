# Three.js Journey — Complete Course Breakdown

Course Creator: Bruno Simon

Official Website: https://threejs-journey.com/

## Course Overview

Three.js Journey is a comprehensive WebGL and Three.js course designed to take students from beginner level to advanced real-time 3D web development.

# Learning Path

The course follows a progressive structure:

1. Learn Three.js Fundamentals
2. Build Interactive 3D Experiences
3. Work with Lighting and Effects
4. Learn Physics and Asset Pipelines
5. Master GLSL Shaders
6. Create Production-Level Scenes
7. Learn React Three Fiber
